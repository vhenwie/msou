# MSOU (Multi-Source Outreach Unfier)

This is a proof-of-concept n8n workflow and related assets for automating sales outreach from multiple data sources by aggregation and data validation.

* n8n
* Hubspot

* Google Sheets
* Gmail
* SMTP (Email)
* Markdown
